## 📖 Sobre o Projeto

A **Evatec** é uma assistente virtual acadêmica desenvolvida para auxiliar alunos e professores no acesso rápido a informações e serviços acadêmicos.

O projeto foi criado com o objetivo de simular um canal de atendimento inteligente capaz de responder dúvidas frequentes, orientar usuários sobre procedimentos acadêmicos e registrar solicitações de forma simples e intuitiva.

A assistente foi desenvolvida como projeto acadêmico, aplicando conceitos de desenvolvimento web, banco de dados, UX e automação de atendimento.

---

## 👩‍💻 Quem é a Evatec?

A **Evatec** é uma personagem virtual criada para representar uma atendente acadêmica digital.

Seu papel é fornecer suporte aos usuários de forma amigável, rápida e eficiente, simulando o atendimento realizado por uma secretaria acadêmica.

### Principais funções

* 📅 Informações sobre calendário acadêmico
* 📄 Solicitação de documentos
* 🎓 Informações sobre matrícula
* 🔄 Trancamento de curso
* 📚 Dúvidas acadêmicas frequentes
* 📌 Registro de protocolos
* 💬 Atendimento automatizado em tempo real

---

## 🚀 Funcionalidades

* Interface de chat interativa
* Botão flutuante para abertura do atendimento
* Respostas automáticas baseadas em palavras-chave

* Experiência semelhante a sistemas reais de atendimento
